# AI Loops — An Iterative Approach to Feature Development

Workshop slides from ODAO 2026 on using AI coding agents in a structured development loop.

## Viewing the Slides

This is a self-contained HTML slide deck with no build step required.

**Option A — Open directly:**

```
open index.html
```

> On macOS this opens in your default browser. On Linux use `xdg-open index.html`, on Windows just double-click the file.

**Option B — Local server (if you hit CORS issues):**

```
npx serve .
```

Then open `http://localhost:3000` in your browser.
